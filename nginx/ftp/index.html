    下载服务器

    ```nginx configuration server { listen 80; server_name xxxxxxxxx; charset utf-8;

    autoindex on;
autoindex_exact_size off;
autoindex_localtime on;


root    /data/download/;
location / {
     fancyindex on;
     fancyindex_exact_size off;
     fancyindex_localtime on;
     fancyindex_header "/myhtml/index.html";
     fancyindex_footer "/myhtml/footer.html";
     fancyindex_css_href "/myhtml/static";
     fancyindex_ignore "myhtml";
     if ($request_filename ~* ^.*?\.(txt|doc|pdf|rar|gz|zip|docx|exe|xlsx|ppt|pptx)$){
        add_header Content-Disposition: 'attachment;';
        }
}

location /static {
     alias /data/download/myhtml/static/;
}

    } ```
